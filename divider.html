Display

Divider

A 1px hairline that separates content into related groups. Dividers exist purely as a visual cue — they carry no semantic meaning, so reach for spacing first and only add a divider when whitespace alone isn't enough to communicate the break.

Accessibility

  • Semantic vs decorative: Use <hr> when the divider marks a meaningful section break (it's announced as "horizontal separator"). For purely decorative cases, use <div role="presentation"> so screen readers don't pause on it.
  • Vertical orientation: If you need a vertical divider in a toolbar, use <div role="separator" aria-orientation="vertical">.
  • Contrast vs decoration: The hairline doesn't need to meet 4.5:1 since it carries no information — but make sure it's visible on whatever surface it lives on (use color/border/interactive on light, swap to a lighter neutral on dark).
  • Don't overuse: Reach for spacing tokens first. A divider on every break creates visual noise; reserve them for places where whitespace alone reads ambiguously.
