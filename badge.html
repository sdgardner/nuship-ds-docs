Badge

Badges are compact, non-interactive labels used to communicate status, category, or metadata at a glance. They pair a semantic color with a short text string to convey meaning without requiring user action.

Properties

PropertyTypeDefaultDescription
label string — Text displayed inside the badge. Rendered uppercase automatically.
color Blue | Green | Purple | Red | Orange Blue Semantic color intent. Maps to Standard, Positive, Neutral, Negative, and Warning respectively.
size Small | Large Small Controls font size (10px vs 12px) and vertical padding (2px vs 4px).
type Outline | Solid Outline Visual treatment — outline uses a tinted background with a colored border; solid uses a filled background with white text.
