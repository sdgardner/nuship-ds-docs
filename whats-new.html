Changelog

Release history

Dated entries for every meaningful change. Tagged so you can scan by intent — added, changed, fixed, removed.

v2.4 · Foundations refresh June 3, 2026
  • Added Site-wide tooltip helper — every page picks up the data-tooltip attribute and renders Skid's .tt-tooltip on hover or focus.
  • Added Design principles page — three pillars (Clarity / Consistency / Scalability) moved out of Foundations and into a dedicated page.
  • Added Coming-soon empty state for Accessibility and Content guidelines, using the Skid Styled Icon component.
  • Changed All headlines on the Foundations page are now sentence case with single-weight Lato 700 — no more mixed 300/900 display flair.
  • Changed Web-token search toolbars share one pop-out treatment: surface-card backdrop, 1px interactive border, soft shadow, 720px max-width.
  • Fixed Hero blue band and the dark-mode "moon" band now extend edge-to-edge across the viewport even on wide screens.
v2.3 · Dark side of the moon June 2, 2026
  • Added Photoreal moon hero on the Foundations dark-mode section — real moon photo wrapped on a lit sphere with a terminator shadow that walks the full lunar phase cycle.
  • Added Floating-card stage on the Foundations hero — 5 translucent token cards reveal in stagger and idle-bob over the blue band.
  • Added Iconography page with ~300 curated Lucide icons in 17 categories, size selector, click-to-copy snippet.
  • Changed Home hero now uses the same blue full-bleed + floating-card pattern as Foundations.
v2.2 · Color & type May 28, 2026
  • Added Interactive Color page — semantic tokens above, primitive ramp below, light/dark mode toggle, click-to-copy.
  • Added Typography, Spacing, Radius, Elevation pages with searchable token tables and live previews.
  • Changed Sticky toolbars across web-token pages now follow the IBM Carbon pop-out pattern (transparent in flow, lifted card when scrolled past the nav).
  • Fixed Contrast on secondary text in the Color page now passes WCAG AA (text-secondary on layer-03 = 7.24:1).
v2.1 · Site shell May 20, 2026
  • Added uShip logo wordmark in the top nav across all 41 pages, with a return-to-home link.
  • Added Landing page (Skid Home Direction 02) — centered hero, component shelf, "Three ways in" entry list.
  • Removed Breadcrumb pattern from the top nav site-wide — the sidebar covers the same need.
