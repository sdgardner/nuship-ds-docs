Tile

Tile is a card-based selection component that pairs a radio or checkbox indicator with visual content, giving users richer context for their choices. Two variants are available: Radio for single-select groups and Checkbox for multi-select groups.

Accessibility

  • Keyboard: Tab moves focus between tiles. Space or Enter toggles or selects. For radio tiles, Arrow keys move selection within the group.
  • ARIA: Radio tiles use role="radiogroup" on the container and role="radio" on each tile. Checkbox tiles use role="group" with role="checkbox". Set aria-checked to match selection state.
  • Focus ring: The 1px solid ring + 4px glow is visible in both themes. Ensure focus is never suppressed on keyboard navigation. Disabled tiles should be removed from the tab order with tabindex="-1".
  • Colour contrast: Selected blue border (#027CB1 light) on white background meets WCAG AA. Label text (#374151 on white) passes 4.5:1. In dark mode, verify lighter blue (#35A8D6) against dark backgrounds.
