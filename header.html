Header

The Header anchors the top of every page with the uShip logo, primary navigation, and user controls. Three variants are available: Shipper for buyer-side flows, Carrier for service-provider flows, and Simple for marketing or signed-out states.

Accessibility

  • Landmark: Wrap the header in <header role="banner"> and the nav in <nav aria-label="Primary"> so assistive tech can jump to it directly.
  • Keyboard: Logo, each nav tab, the user menu trigger, and the help icon must all be reachable in tab order. Enter activates them; Esc closes any opened menu.
  • Skip link: Render a "Skip to main content" link as the first focusable element so keyboard users can bypass the nav.
  • Contrast: Tab text (#374151 on white) and the link blue (#016692) both clear WCAG AA. Verify the equivalent dark-mode tokens before shipping.
