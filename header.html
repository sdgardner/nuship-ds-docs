Header

The Header anchors the top of every page with the uShip logo, primary navigation, and user controls. Three variants are available: Shipper for buyer-side flows, Carrier for service-provider flows, and Simple for marketing or signed-out states.

Usage Guidelines

Brand identity
  • Place the logo at the top-left where users expect to find it
  • Link the logo to the homepage or primary dashboard
  • Use the official brand mark — never substitute for a generic icon
Primary navigation
  • Surface 3–7 top-level routes; collapse the rest into a menu
  • Indicate the active route with a clear visual treatment
Persistent placement
  • Keep the header consistent across every page in the same product context
  • Use sticky positioning only when the navigation is needed during long scrolls
Account & utilities
  • Place account, search, and global actions on the right side
  • Use icons paired with accessible labels for compact utility actions
Compete with content
  • Don't make the header so tall it crowds the page content below
  • Don't apply heavy colors or shadows that overpower the main UI
Overload with options
  • Don't list more than 7 nav items at the top level — move overflow into a menu
  • Don't mix marketing CTAs with product navigation in the same row
Hide on scroll quirks
  • Don't auto-hide the header during normal scrolling — it disorients users
  • Don't change the header's height as the user scrolls; keep dimensions stable
Lose context on mobile
  • Don't collapse the logo or brand into the hamburger menu
  • Don't drop the active-route indicator at smaller breakpoints

Accessibility

  • Landmark: Wrap the header in <header role="banner"> and the nav in <nav aria-label="Primary"> so assistive tech can jump to it directly.
  • Keyboard: Logo, each nav tab, the user menu trigger, and the help icon must all be reachable in tab order. Enter activates them; Esc closes any opened menu.
  • Skip link: Render a "Skip to main content" link as the first focusable element so keyboard users can bypass the nav.
  • Contrast: Tab text (#374151 on white) and the link blue (#016692) both clear WCAG AA. Verify the equivalent dark-mode tokens before shipping.
