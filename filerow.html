File Row

A horizontal row that represents a single file inside an upload list. Each row pairs a status-coloured icon tile, the filename, an inline progress bar, and a context-appropriate trailing action (cancel for in-progress, success check for complete). Four states are supported — Default (queued), Uploading, Success, and Error.

Accessibility

  • ARIA: Mark the row's progress with role="progressbar", aria-valuenow, aria-valuemin="0", and aria-valuemax="100" so the current percent is announced.
  • Status announcements: Wrap the upload list in aria-live="polite" so transitions from Uploading → Success or Error are read out without interrupting other content.
  • Don't rely on colour alone: Status colour is paired with text ("Queued", "Uploading…", "Complete", "Failed") and an icon shape change so colour-blind users still get the state.
  • Keyboard: The cancel / remove button is reachable via Tab and activated by Enter or Space. The success-state checkmark is decorative — give it aria-hidden="true".
