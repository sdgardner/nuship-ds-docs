Segmented Control

A Segmented Control presents a set of mutually exclusive options as a horizontal strip of connected buttons. Selecting an option immediately updates the visible content. It supports 2, 3, or 4 segments.

Accessibility

  • Role: Use role="tablist" on the container and role="tab" on each segment. Set aria-selected="true" on the active segment and aria-selected="false" on all others.
  • Keyboard: Arrow Left/Right moves focus between segments and selects the focused one. Home jumps to the first segment, End to the last.
  • Focus ring: The 1px solid ring + 4px glow is visible in both themes. Ensure focus is never suppressed on keyboard navigation.
  • Colour contrast: White text (#FFFFFF) on the selected fill (#027CB1 light) meets WCAG AA at 4.6:1. In dark mode the lighter blue (#35A8D6) requires verification against the white text.

Related Components

Tabs
Use Tabs when the options control separate content panels. Segmented Control is for switching a view mode within one panel.
Radio Button
Use Radio Buttons when the options need descriptions or a vertical layout. Segmented Control is compact and horizontal.
Dropdown
Use a Dropdown when there are more than 4 options or the labels are too long for inline display.
