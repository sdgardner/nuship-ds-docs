Metrics

Metric cards surface a single key number alongside a brief title and a percentage delta. The delta colour signals direction — green for an increase and red for a decrease — so the trend reads at a glance.

Usage Guidelines

Pair value with context
  • Always show a clear label alongside the numeric value
  • Include units, currency, or timeframe so the value is unambiguous
  • Use a short helper line for definitions or methodology when needed
Use trends meaningfully
  • Pair a delta or trend indicator with a comparison window (e.g. "vs last week")
  • Use semantic color (green/red) only when up/down has clear meaning
Group related metrics
  • Place related KPIs side-by-side so users can compare them at a glance
  • Limit a single row to 3–5 metrics — more becomes overwhelming
Keep formatting consistent
  • Round numbers to the same precision across a metric group
  • Abbreviate large values consistently (e.g. 1.2K, 3.4M)
Show raw numbers without context
  • Don't display a value without a label, unit, or timeframe
  • Don't truncate large numbers so much that they lose meaning
Mix unrelated KPIs
  • Don't group metrics from different domains in the same row
  • Don't pair conflicting units (currency next to count next to percent) without clear visual separation
Overuse trend color
  • Don't show a red or green delta on every metric — it dilutes meaning
  • Don't assume up always means good (e.g. higher error rate is bad)
Use as a data table
  • Don't use Metrics to display long lists of records — use a table instead
  • Don't show more than ~10 metrics on a single screen

Accessibility

  • Don't rely on colour alone: The delta uses both colour (green/red) and a +/− prefix so users with low vision or colour-blindness still get the trend.
  • Screen reader announcement: Compose a single accessible label like "Metric Title: 1234, up 15%" on the wrapping element so the card reads as one coherent statement, not three disjoint values.
  • Number formatting: Display values with appropriate locale formatting (thousands separators, currency symbols). For long numbers, abbreviate (e.g. 1.2M) but keep the full value in the accessible label.
  • Direction semantics: Up isn't always good. Flip the colour mapping for metrics where a decrease is desirable (churn, errors, p95 latency).

Related Components
