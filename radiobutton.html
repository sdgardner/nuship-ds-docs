Radio Button

A Radio Button allows the user to select exactly one option from a mutually exclusive set. Each radio shows a label and optional description text, providing context for the choice.

Accessibility

  • Role: Use a native <input type="radio"> element so the browser provides role="radio" and aria-checked automatically. Group radios with a <fieldset> and <legend> to describe the set.
  • Keyboard: Tab moves focus into the radio group and to the selected item (or the first item if none is selected). Arrow Up/Down cycles through and selects options. Space selects the focused radio if not already selected.
  • Focus ring: The 4px glow ring meets WCAG 2.4.7 (Focus Visible). Ensure the glow is never suppressed and is visible against the page background in both light and dark themes.
  • Labels: Each radio must have a visible, associated <label>. Description text is connected via aria-describedby so screen readers announce both the label and the description when the radio receives focus.

Related Components

Checkbox
Use a Checkbox when the user can select multiple options. Use Radio when exactly one choice is allowed.
Dropdown
Use a Dropdown when the set of options is long (>5) and a radio group would take too much vertical space.
Switch
Use a Switch for a binary on/off toggle. Use Radio when there are three or more mutually exclusive choices.
