Iconography

nuShip uses Lucide — an open-source, MIT-licensed icon library with a unified 24×24 grid, 1.5px stroke, and rounded line caps. Every icon ships as a tree-shakeable SVG; the same icon name renders identically across web, native, and Figma.

Browse the full set at lucide.dev/icons. Click any tile below to copy the HTML snippet.

Drop-in usage

Reference an icon by name with data-lucide="<name>". Lucide auto-renders any <i> tag with that attribute when lucide.createIcons() runs.

<script src="https://unpkg.com/lucide@latest"></script> <i data-lucide="package" style="width:20px;height:20px"></i> <script>lucide.createIcons();</script>
